# Langchain_Tutorials
LangChain Tutorial – A beginner friendly guide to building LLM-powered applications. This repo covers core concepts: models, prompts, chains, memory, agents &amp; retrieval-augmented generation (RAG). Learn to integrate OpenAI, Hugging Face &amp; tools like vector databases. Includes hands-on Python examples for chatbots, document Q&amp;A &amp; web search agents.
